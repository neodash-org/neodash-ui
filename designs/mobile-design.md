## Mobile-Specific Features

### Gesture Controls
- **Swipe Left/Right**: Navigate between tabs
- **Pull to Refresh**: Update portfolio data
- **Pinch to Zoom**: Enlarge charts
- **Long Press**: Quick actions menu

### Touch Optimizations
- **Minimum Touch Target**: 44px × 44px
- **Spacing**: 8px minimum between interactive elements
- **Feedback**: Haptic feedback for important actions
- **Loading States**: Skeleton screens for better UX

### Performance Considerations
- **Lazy Loading**: Load charts and data on demand
- **Image Optimization**: Compressed images and SVGs
- **Caching**: Offline support for basic data
- **Progressive Enhancement**: Core functionality works without JavaScript

## Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px+
